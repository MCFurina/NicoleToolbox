# NicoleToolbox 尼可工具箱
一个原神工具箱  
使用 .NET 10.0 和 WinUI3 制作  


# 如何使用？
支持 Windows 10 和 Windows 11，推荐使用 Windows 11 以获得更好的体验。  
使用前请先下载并安装 .NET 10.0！  
[前往 .NET 10.0 下载界面](https://dotnet.microsoft.com/zh-cn/download/dotnet/10.0)  
SDK 和 .NET 桌面运行时均可运行本应用！但是更推荐安装 SDK，因为功能完整，包含开发者工具。  

应用压缩文件可从 GitHub Releases 下载！  
下载完成后，解压文件，双击 NicoleToolbox.exe 即可运行！
